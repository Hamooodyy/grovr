# Grovr

Grovr is a grocery price comparison MVP that takes the guesswork out of price comparison shopping. Grovr analyzes prices for a user's cart across nearby grocery stores, and recommends placing orders at the location that meets their brand, delivery, and pricing criteria

## Features

- **Grocery list builder** — add items one at a time, manage quantities
- **Nearby store lookup** — finds stores (Kroger, Ralphs, Fred Meyer, Harris Teeter, etc.) within a user-defined radius
- **Real-time price comparison** — fetches estimated prices from per item per store; promo prices are always preferred
- **Lowest-cost recommendation** — highlights the cheapest store with an estimated subtotal
- **Instacart handoff** — users are taken to a pre-filled instacart cart where they can place their order
- **Authentication** — sign-up and sign-in via Clerk
